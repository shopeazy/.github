## ✨ What is ShopEazy?

ShopEazy is an **all-in-one e-commerce platform built for South African entrepreneurs**. Whether you're selling handmade crafts, fashion, electronics, or anything in between, ShopEazy gives you everything you need to grow a thriving online business — without the complexity.

> *Resilient. Bold. Built for Africa.*

| Feature | Description |

## 🎨 Design Philosophy

ShopEazy is built with **"SA Brutalism"** — a design language that is resilient, bold, and energetic, made for the South African market.

- **Zero rounding** — sharp, confident edges
- **Hard shadows** — solid offset shadows for depth
- **Brand colours** — Navy `#101935`, Pink `#FF5275`, Orange `#FFA62B`, Teal `#16697A`
- **Mobile-first** — South Africa is a mobile-first market, and so are we
